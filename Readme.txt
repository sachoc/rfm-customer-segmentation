# Customer Segmentation for E-Commerce Using RFM Analysis

## 🔍 Objective
Segment customers using RFM (Recency, Frequency, Monetary) metrics to help businesses target different groups with appropriate marketing strategies.

## 📁 Dataset
- Source: UCI Online Retail Dataset
- File: `Online_Retail.xlsx`

## 🛠️ Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📊 Key Steps
- Clean and preprocess the data
- Compute RFM metrics
- Score customers based on quartiles
- Segment them with custom logic
- Perform KMeans clustering
- Visualize segments and clusters

## 📦 Outputs
- `RFM_Segments_Final_Labeled.xlsx`: Final labeled dataset
- `.png` visualizations: Segment distribution, RFM scatter, and KMeans clustering
- `.ipynb`: Full Python code with results
